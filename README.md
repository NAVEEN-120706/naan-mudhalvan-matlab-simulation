# Naan Mudhalvan – MATLAB Simulation for EV Technology

## Objective
This project contains MATLAB/Simulink simulations related to Electric Vehicle (EV) technology
and power electronics systems used in EV manufacturing.

## Software Used
- MATLAB
- Simulink


## Simulations Included
1. CUK Converter
2. Single Phase Inverter
3. Buck Converter
4. Boost Converter
5. Buck-Boost Converter
6. Battery Model
7. SOC Estimation using Coulomb Counting
8. Active Cell Balancing
9. Passive Cell Balancing
10. BLDC Motor Back EMF Model

## How to Run
1. Download or clone this repository
2. Open MATLAB
3. Open required .slx file in Simulink
4. Click on Run (▶)
5. Observe output in Scope blocks

## Application in EV Manufacturing
These simulations represent key systems used in electric vehicles such as:
- Battery Management System (BMS)
- Motor drive system
- Power converters
- Energy management
